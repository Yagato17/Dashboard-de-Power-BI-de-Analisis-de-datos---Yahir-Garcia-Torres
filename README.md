# Dashboard Interactivo de Ventas:
Diseño de un tablero de inteligencia de negocios para el monitoreo de indicadores clave (KPIs) y análisis geográfico de ventas.

### Desarrollo Técnico
* **Power Query**: Se utilizó para la extracción, transformación y carga (ETL) de los datos, asegurando una estructura óptima para el modelo.
* **Visualización Geográfica**: Implementación de mapas interactivos para localizar la densidad de ventas por ciudad y país.
* **Análisis Temporal**: Creación de gráficas de tendencia para comparar el rendimiento de ventas año tras año.

### 💡 Insights Clave
* Identificación del **Top 5 de productos** con mayor facturación.
* Análisis de participación por **categoría de producto**.
* Visualización global del mercado con filtros segmentados por empleados y clientes.
