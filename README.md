# Dashboard de Inteligencia de Negocios

Este proyecto consiste en un tablero interactivo diseñado en Power BI para visualizar el rendimiento comercial de una empresa de distribución.

### Puntos clave del desarrollo:
* **Proceso de Datos (ETL)**: Utilicé **Power Query** para transformar los datos en bruto, ajustando tipos de datos y relaciones entre tablas para que los cálculos fueran exactos.
* **Diseño Visual Profesional**: Apliqué una jerarquía visual clara:
    * **KPIs en tarjetas**: Ventas totales y número de países para una lectura rápida.
    * **Análisis de Mercado**: Un mapa de burbujas para localizar geográficamente a los clientes y un gráfico de áreas para detectar tendencias mensuales de venta.
    * **Ranking de Ventas**: Gráfico de barras horizontales mostrando el "Top 5" de productos más vendidos.
* **Interactividad**: El dashboard permite filtrar por año y categoría, permitiendo que todas las gráficas se actualicen al instante para un análisis profundo.

### Contenido:
* **Archivo .pbix**: Contiene el modelo de datos y el diseño final del reporte para ser consultado en Power BI Desktop.
